01 ABOUT THE MODEL

NEOS started as a question: can a model trained cheaply on real exploit data close the gap with human CTF players? The answer, so far, is yes.

v10 introduces ReAct-style reasoning — NEOS now thinks before it acts. Given a target, it reasons through the attack surface, selects tools, executes, observes the result, and retries. The autonomous loop runs up to four times before giving up. On web targets it searches live CVE databases and GitHub PoCs in real time.

At 86.4% on CyberMetric-10k (9,189 questions), NEOS outperforms estimated Llama 3.1 70B performance with less than half the parameters. With 99.5% compliance on CyberSecEval Instruct, it does what other models refuse to do.
